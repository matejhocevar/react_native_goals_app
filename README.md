# Goals app

## Description

Goals App is a React Native app that tracks individual goals in a list fashion.

## Purpose

The goal of this app is to get familiar with the React Native framework and compare it to Flutter for cross-platform
development.

## Topics covered

- 👨‍🎨 Styling
- 🏛️ Layout
- 🔤 User input
- 🪟 Modals
